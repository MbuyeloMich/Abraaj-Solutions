# Abraaj-Solutions

A starter repository for Abraaj Solutions — skeleton project for collaboration.

## Quick start

Clone the repository and add files or edits locally, then commit and push:

```bash
git clone git@github.com:MbuyeloMich/Abraaj-Solutions.git
cd Abraaj-Solutions
# add or edit files
git add README.md LICENSE .gitignore
git commit -m "docs: ✨ Add README, LICENSE, .gitignore"
git push origin main
```
